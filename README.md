# Tekstowa Gra RPG w C++ ⚔️
**Projekt Zaliczeniowy - Programowanie Obiektowe w C++**

Projekt to autorski, konsolowy silnik gry RPG, demonstrujący praktyczne wykorzystanie zaawansowanych mechanizmów języka C++ (Standard Modern C++). Gracz może stworzyć własnego bohatera: Wojownika lub Maga, walczyć z przeciwnikami i gromadzić ekwipunek, a stan rozgrywki jest trwale zapisywany na dysku.

## Instrukcja Kompilacji
Projekt został podzielony na moduły. Aby skompilować grę z poziomu terminala, użyj kompilatora wspierającego standard C++11 (lub nowszy), np. `g++`:

```bash
g++ *.cpp -o gra
./gra
